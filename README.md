# 坐标转换工具

网页工具，支持多种坐标系统转换：GPS 坐标（WGS84）与中国地图坐标（GCJ02）、平面坐标系转换、度分秒格式互转。

Developed with *Claude* and *ChatGPT*

用*Claude*及*ChatGPT*开发

## 功能模块

### 火星坐标系转换
- WGS84 ⇄ GCJ02：GPS 原始坐标与中国地图坐标互转
- 支持单条和批量转换
- 有效范围：中国大陆地区（经度 73.56°~135.05°，纬度 3.51°~53.56°）

### 平面坐标系转换
- 广州2000平面坐标 → CGCS2000 经纬度
- 广州平面坐标 → 西安80平面坐标（四参数变换）→ CGCS2000 经纬度
- 西安80椭球采用 IAG-75，最终投影为 GRS80（CGCS2000）

### 度分秒转换
- 支持小数、度分、度分秒三种格式互转
- 自动识别输入格式（无需手动选择）
- 支持多种分隔符：° ′ ″ 或空格/逗号

## 使用说明

### 火星坐标转换
1. 点击「火星坐标系转换」主标签
2. 单条转换：输入经纬度，选择转换方向，点击对应按钮
3. 批量转换：粘贴多条坐标（格式：经度,纬度 或 经度 纬度），点击转换
4. 支持格式转换：小数形式、度分秒、度分

### 平面坐标转换
1. 点击「平面坐标系转换」主标签
2. 选择转换类型：广州2000 或 广州平面
3. 输入 X、Y 坐标值
4. 点击「转换为CGCS2000经纬度」

### 度分秒转换
1. 点击「度分秒转换」主标签
2. 输入角度数据（格式自动识别）
3. 点击「自动转换」
4. 查看三种格式的转换结果

### 快捷键
- 在输入框按 `Enter` 直接执行转换

## 注意事项

- 坐标转换存在小数点级别的精度损失（正常现象）
- 火星坐标转换仅适用于中国大陆地区
- 平面坐标转换假设无七参数平移（实际应用需验证）
- 本工具 100% 本地运行，无服务器通信
- 所有数据处理在浏览器完成，无隐私风险
- 支持离线使用
