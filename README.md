# 坐标转换工具

网页工具，支持多种坐标系统转换：GPS 坐标（WGS84）与中国地图坐标（GCJ02）、平面坐标系转换、度分秒格式互转，以及高程转换。

Developed with *Claude* and *ChatGPT*

用*Claude*及*ChatGPT*开发

## 功能模块

### 火星坐标系转换
- WGS84 ⇄ GCJ02：GPS 原始坐标与中国地图坐标互转
- 支持单条和批量转换
- 有效范围：中国大陆地区（经度 73.56°\~135.05°，纬度 3.51°\~53.56°）

### 平面坐标系转换
- 广州2000平面坐标 → WGS84 经纬度
- 广州平面坐标 → 西安80平面坐标（四参数变换）→ WGS84 经纬度
- 西安80椭球采用 IAG-75，最终输出按 WGS84 椭球参数计算（未引入七参数）

### 度分秒转换
- 支持小数、度分、度分秒三种格式互转
- 自动识别输入格式（无需手动选择）
- 支持多种分隔符：° ′ ″ 或空格/逗号

### 高程转换
- 经纬度支持小数 / 度分 / 度分秒自动识别
- 单一高度输入框，支持按“输入类型 → 目标类型”一键转换
- 输入/目标类型支持：GPS高度、1985国家高程系、广州高程基准、1956黄海高程系、珠江高程基准
- 支持可选自定义差值
- 基于 EGM2008 模型进行换算

## 使用说明

### 火星坐标转换
1. 点击「火星坐标系转换」主标签
2. 单条转换：输入经纬度，选择转换方向，点击对应按钮
3. 批量转换：粘贴多条坐标（格式：经度,纬度 或 经度 纬度），点击转换
4. 支持格式转换：小数形式、度分秒、度分

### 平面坐标转换
1. 点击「平面坐标系转换」主标签
2. 选择转换类型：广州2000 或 广州平面
3. 输入 X、Y 坐标值
4. 点击「转换为WGS84经纬度」

### 度分秒转换
1. 点击「度分秒转换」主标签
2. 输入角度数据（格式自动识别）
3. 点击「自动转换」
4. 查看三种格式的转换结果

### 高程转换
1. 点击「高程转换」主标签
2. 输入经纬度（支持小数 / 度分 / 度分秒）
3. 输入一个高度值
4. 选择输入类型和目标类型
5. 点击「开始转换」

### 快捷键
- 在输入框按 `Enter` 直接执行转换

## 注意事项

- 坐标转换存在小数点级别的精度损失（正常现象）
- 火星坐标转换仅适用于中国大陆地区
- 平面坐标转换假设无七参数平移（实际应用需验证）
- 高程转换依赖在线接口（`https://height-translate.waku.icu`）
- 高程转换基于 EGM2008 模型，不保证米级以下精度
- 除高程转换外，其余转换在浏览器本地完成
